#  Mess Feedback System

##  Problem
Students often face issues with mess food quality but lack a structured way to provide feedback.

##  Solution
This project provides a web-based platform where students can:
- Submit feedback
- Rate meals
- View all feedback
- See average ratings

##  Features
- Feedback form (name, meal, rating, comment)
- Displays all feedback
- Average rating calculation
- Delete feedback option
- Data stored using localStorage

##  Tech Stack
- HTML
- CSS
- JavaScript

##  How to Run
1. Download the project
2. Open `index.html` in your browser

##  Learning Outcomes
- DOM manipulation
- Local storage handling
- Event handling in JavaScript
- UI design basics
